Keyboard Shortcuts

"Ctrl-Enter": Run
"Ctrl-Space": AutoComplete
"Ctrl-I": Show Type
"Ctrl-O": Show Docs
"Alt-.": Jump to Definition
"Alt-,": Jump Back
"Ctrl-Q": Rename
"Ctrl-.": Select Name

"Shift-Ctrl-C": Toggle Comments on selection
"Ctrl-Slash": Comment Selection
"Shift-Ctrl-Slash": Uncomment Selection
